Stacking of phases in RGB Channel Direction

Phases are stacked in the RGB channel Direction (X,Y,Z,C)
Pre contrast is Red
Peak Enhancement is Green
Delayed Enhancement if Blue
Allows for easier confirmation of proper spatial and intensity normalization

Multichannel Intensity Normalization

A 95% winsorization is performed
Img>percentile(img,97.5) = percentile(img,97.5)
ImgThe exam is then normalized to the highest signal intensity in the exam by dividing the exam by the maximum voxel value.
Img = img/img.max()

Stacking DCE-MRI phases in the RGB color channels produces a visually striking difference between the typical washout, plateau, and persistent uptake enhancement dynamics commonly displayed by breast tumors, facilitating visual confirmation of proper intensity normalization for large datasets.
